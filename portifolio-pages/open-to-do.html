Portfoio Details

  1. Home
  2. OPEN TO DO

    For a modern lock system

  1. Password Entry into the Application: The first step involves the user entering a secure password directly into the application on their smartphone. This app is connected to your home's digital door lock via a secure network.

  2. Signal Sending: After the password is entered, the application sends an encrypted signal to the digital lock. This signal is protected by advanced security protocols to prevent unauthorized interception or access.

  3. Confirmation and Unlock: The digital lock receives the signal and verifies the password. If the password is correct, the lock activates the unlocking mechanism, allowing access to the residence. This process ensures that only people with an authorized password can enter.

  4. Access Notification: After the door is unlocked, the app sends a notification back to the user's smartphone, confirming that access has been granted. This not only informs the user that the door was opened correctly, but also serves as a record of when the door was accessed.
